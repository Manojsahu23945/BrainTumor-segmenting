# 🧠 Brain Tumor Segmentation Using Multi-Scale Attention U-Net

## Overview

This project focuses on brain tumor segmentation from MRI images using a Multi-Scale Attention U-Net with an EfficientNetB4 encoder. The model is designed to accurately identify tumor regions while maintaining computational efficiency through transfer learning and attention mechanisms.

## Features

- Brain MRI image segmentation
- EfficientNetB4 pretrained encoder
- Multi-Scale Attention U-Net architecture
- Image preprocessing using CLAHE and Gaussian filtering
- Tumor mask prediction
- Performance evaluation using Dice Score, IoU, Precision, Recall, and Specificity

## Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- EfficientNetB4
- U-Net
- Kaggle GPU

## Dataset

- LGG Brain Tumor Segmentation Dataset
