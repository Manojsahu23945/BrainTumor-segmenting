# Language Agnostic Chatbot

An AI-powered multilingual chatbot that enables users to interact in their preferred language and receive accurate responses from institutional documents using Retrieval-Augmented Generation (RAG).

## Features

- Multilingual Support
- AI-Powered Responses
- Document-Based Question Answering
- PDF Knowledge Base
- Voice Input & Output
- Chat History
- Admin Dashboard
- WhatsApp & Telegram Integration

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js
- PostgreSQL
- Prisma

### AI
- FastAPI
- LangChain
- Gemma-2
- IndicTrans2
- ChromaDB

## Architecture

User → React Frontend → Express API → FastAPI AI Service → ChromaDB/Gemma-2
